# 登陆系统设计(新用户/旧用户 适配)
>  app.js程序加载时
1. 登陆启动(重新登陆)
  - 调用微信登陆
    - 成功返回
    - 启动服务器登陆
    - 返回会话信息(token,uId)
2. 更新用户信息或初始化用户信息
    - 获取用户信息权限
    - 用户信息存服务器:新用户插入,旧用户更新,同时存储本地(头像,昵称)
3. 进入主页
